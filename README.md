
# RestaurantIQ

RestaurantIQ is a restaurant back-of-house operations platform designed to help restaurants manage inventory, purchasing, and supplier invoices in one system.

## Features

- Inventory management and counting
- PAR level management
- Smart ordering based on current stock
- Purchase order tracking
- Invoice parsing and review
- Supplier delivery discrepancy detection
- Waste tracking
- Operational alerts

## Tech Stack

Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

Backend
- Supabase (Postgres, Auth, Storage)
- Edge Functions

State Management
- TanStack Query

Charts
- Recharts

## Core Workflow

Inventory → Smart Order → Purchase Order → Supplier Invoice → Inventory Update → Waste Tracking
